# 👁️ Retina Disease Detection using EfficientNet-B0

An AI-powered web application for automated retinal disease detection using **EfficientNet-B0** and **Streamlit**. This project classifies retinal fundus images to assist in the early detection of retinal diseases using deep learning.

---

## 📌 Project Overview

Retinal diseases can lead to vision impairment if not detected early. This project leverages the power of **Deep Learning** to classify retinal fundus images and provide quick predictions through an interactive web interface.

The application is built using **Python**, **PyTorch**, **EfficientNet-B0**, and **Streamlit**, making it easy to use for demonstration and educational purposes.


## 📷 How It Works

1. Upload a retinal fundus image.
2. The image is preprocessed.
3. EfficientNet-B0 extracts image features.
4. The trained model predicts the retinal disease class.
5. The predicted result is displayed on the Streamlit interface.
